Banshee – Privacy Policy Last updated: April 2026 Banshee is a browser extension that detects whether the developer studio of a game (on Steam or Instant Gaming) is based in Europe. It is developed independently, with no commercial purpose. Data collection Banshee does not collect, store, transmit or sell any personal data. No analytics, no tracking, no third-party services. Unknown studio reporting When a studio is not found in Banshee's database, its name is automatically and anonymously sent to a private Discord channel to help improve the database. No user data, URL, browser history or any identifying information is attached to this report — only the raw studio name as it appears on the page. Web page content Banshee reads the text content of game pages (Steam and Instant Gaming) solely to extract the developer studio name. This information is processed locally in your browser and is never sent to any external server. User preferences Your settings (language, popup preferences) are saved locally using chrome.storage.local. They never leave your device. External resources Banshee loads country flag images from flagcdn.com, a public CDN, for display purposes only. No user data is included in these requests. Contact Questions? Reach out via the Banshee Discord. Banshee · Independent project · No ads · No data collection
